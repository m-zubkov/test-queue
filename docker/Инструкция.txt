1. Клонировать репозиторий на локальную машину

2. В папке docker скопировать файл .env.sample в .env

3. В папке config скопировать файл db-local.php.sample в db-local.php

4. Перейти в папку docker и последовательно выполнить команды
	make composer-install
	make up
	make migrate

5. Для генерации событий в очереди можно воспользоваться одной из команд
	make generate
	make generate-alternative
   
   Каждая из них генерит 10000 событий, разница лишь в последовательности генерации по аккаунтам

6. Для запуска обработки выполнить команду
	make process

7. Проверить результат выполнения командой
	make log account_id=1

   Такая команда выведет лог обработки событий для аккаунта #1
